### Website

The website is  located in the ./site folder. A  static (HTML) copy of
the site  can be  found in ./site/_site.  It requires the  jekyll Ruby
gem.  To build it, run

    cd site jekyll

To make  a page on the  site, just create  a new markdown file  in the
site folder.
